# Mercart

Mercart is a full-stack e-commerce web application that allows users to register, log in, browse and search products by category, manage a wishlist, add items to a shopping cart, and place orders.

---

## 🔗 Live Website

- **Website Link**: [https://mercart-pi.vercel.app](https://mercart-pi.vercel.app)

---

## 🛠️ Tech Stack

- **Frontend**: React, Vite, React Router, CSS
- **Backend**: Node.js, Express.js
- **Database**: MySQL (Hosted on Clever Cloud)
- **Authentication**: JSON Web Tokens (JWT) secured with HTTP-Only Cookies
- **Deployment**: Vercel (Frontend) & Render (Backend) & Db (Clever Cloud)
